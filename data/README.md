# Data

This project keeps large public ad datasets out of Git.

## Avazu CTR data

Download the Avazu Click-Through Rate Prediction dataset from Kaggle and place the raw file here:

```text
data/raw/avazu/train.csv
```

Then run:

```bash
python3 scripts/run_real_avazu.py --input data/raw/avazu/train.csv --nrows 100000
```

Avazu contains impression-level click labels but not conversion or clearing-price labels. In this project it is used to train a real CTR model; the auction cost, win/loss, and post-click conversion feedback are still generated by the simulator so strategy experiments remain end-to-end runnable.
